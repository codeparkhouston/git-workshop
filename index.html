What we'll do

GitHub

Get your first repo onto GitHub following along with GitHub's Hello World. No coding required!

Git

Dig in with the commandline as we learn about using git to get a project from our computers onto GitHub.

Who's coming?

Learners and mentors from Houston's diverse tech meet ups including: R User Group, Girls' Coding Club, Data Viz, and much more!

Together, we'll share how we use code in so many different exciting ways while learning Git and GitHub.

All levels of experience welcome!

Why?

GitHub

GitHub is a great tool for collaborating on and contributing to open source projects. It's the place to go to find code online for all kinds of cool things.

Git

Git helps track the history of your project as you try different versions and work on different features with others. GitHub was built for hosting your Git projects.

What do I need to bring?

You'll need a laptop and a semi-full tummy. We'll have snacks and drinks there to top you off.
