Why?

GitHub

GitHub is a great tool for collaborating on and contributing to open source projects. It's the place to go to to find code online for everything from serious scientific code to web frameworks to all kinds of cool things.

Git

Git helps track the history of your project as you try different versions and work on different features with others! It gives you the power to have parallel universes (of code) and time travel (through the history of your code) to change it. GitHub was built for hosting your Git projects.
